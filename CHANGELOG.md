# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- Added public-facing contribution, citation, code-of-conduct, and changelog
  metadata.
- Added a public README orientation for graph-RC scope, related repositories,
  quick start, repository map, reproducibility expectations, and license/citation
  boundaries.
- Added a finite publication checklist for GitHub-readiness work.
- Added a README claim-to-evidence table and tracked quickstart visual assets.
- Added an experiment reconstruction index under `experiments/README.md`.
- Added a quickstart smoke test that verifies telemetry and visual artifacts.
- Added release notes for the first public GitHub snapshot.

### Changed

- Clarified output policy: top-level scratch outputs remain ignored, while
  experiment-local outputs may be retained as historical evidence and must use
  relative paths.
- Clarified mixed license and citation wording across README metadata,
  `CITATION.cff`, and the publication checklist.
- Tightened publication-review follow-ups: aligned package/citation authorship,
  made `CITATION.cff` use a single root SPDX license, clarified the LGRC event
  queue path in the README diagram, and added known-gap/post-publication
  checklist sections.
