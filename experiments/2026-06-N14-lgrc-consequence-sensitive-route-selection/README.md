# N14 - LGRC Consequence-Sensitive Route Selection

N14 is an agency-prerequisite experiment. It starts after the N13
support-seeking regulation closeout and asks whether route selection can depend
on recorded downstream consequences for support, memory, or regulation rather
than only on immediate route affordance.

The core question is:

```text
Can route selection depend on expected downstream effects on support, memory,
or regulation, rather than immediate route affordance alone, while preserving
claim boundaries, replay, and budget controls?
```

N14 does not prove agency. It does not prove intention, semantic choice,
semantic goal ownership, identity acceptance, selfhood, personhood, biological
behavior, native support, or fully native agentic-like integration.

## Hypotheses

Hypothesis A:

```text
Route candidates can be annotated with source-backed, pre-selection
consequence records over support, memory, and regulation surfaces without
using hidden outcome tables or post-hoc consequence scoring.
```

Hypothesis B:

```text
Route selection can be classified as consequence-sensitive when the selected
route changes with recorded downstream support, memory, or regulation effects
under replayable budget constraints, rather than immediate affordance or
fixture labels alone.
```

Hypothesis C:

```text
Even if consequence-sensitive selection is observed, intention, agency,
semantic choice, semantic goal ownership, identity acceptance, selfhood,
personhood, biological behavior, native support, fully native integration, and
unrestricted agency remain blocked unless separate theory and native
implementation gates are satisfied.
```
