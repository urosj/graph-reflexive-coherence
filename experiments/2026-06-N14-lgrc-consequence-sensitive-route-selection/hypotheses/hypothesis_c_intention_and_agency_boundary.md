# Hypothesis C - Intention And Agency Boundary

## Statement

```text
Even if consequence-sensitive selection is observed, intention, agency,
semantic choice, semantic goal ownership, identity acceptance, selfhood,
personhood, biological behavior, native support, fully native integration, and
unrestricted agency remain blocked unless separate theory and native
implementation gates are satisfied.
```

## N14-Specific Meaning

N14 is close to intention-facing language because it asks whether routes are
selected by expected downstream effects. This hypothesis tests whether that
result stays inside the AP4 agency-prerequisite boundary.

The N14 positive result, if achieved, can be only:

```text
artifact_level_ap4_consequence_sensitive_route_selection_candidate
```

not:

```text
intention
agency
semantic choice
semantic goal ownership
identity acceptance
selfhood
personhood
biological behavior
native support
fully native agentic-like integration
```
