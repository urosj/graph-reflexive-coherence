# Hypothesis B: Generalization Envelope

Hypothesis B asks whether the N11 transfer results form a meaningful envelope:
what survives, what fails, and why.

## Question

```text
Does the integration pattern have a bounded reusable regime across context,
support, proxy, and longer-window variation, or is it only a source-specific
N10 construction?
```

## Required Evidence

```text
positive transfer rows
negative transfer rows
matrix of context/support/proxy variants
longer-horizon trend records
failure blockers with distinct reasons
artifact-only replay
claim-clean closeout
```

## Interpretation Rule

Hypothesis B should not chase universal success. It should classify:

```text
surviving lanes
bounded degradation
support-sensitive failure
proxy-envelope failure
context-transfer failure
stale-source failure
native-policy gap
```

The envelope is the result.
