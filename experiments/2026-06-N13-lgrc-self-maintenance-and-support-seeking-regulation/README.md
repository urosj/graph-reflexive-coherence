# N13 - LGRC Self-Maintenance And Support-Seeking Regulation

N13 is an agency-prerequisite experiment. It starts after the N12 bridge
closeout and asks whether artifact-level LGRC composition can regulate toward
source-current support conditions rather than only an externally declared
proxy.

The core question is:

```text
Can the system regulate to preserve support conditions derived from recorded
support state, rather than only an externally declared proxy, while preserving
claim boundaries, replay, and budget controls?
```

N13 does not prove agency. It does not prove intention, semantic goal
ownership, identity acceptance, selfhood, personhood, biological behavior, or
fully native agentic-like integration.

## Hypotheses

Hypothesis A:

```text
Source-backed support-state evidence can be inventoried into candidate support
conditions without promoting support survival into identity acceptance.
```

Hypothesis B:

```text
Support-seeking regulation can be distinguished from external proxy regulation
when the target condition is derived from source-current support state,
serialized, replayable, budgeted, and control-clean.
```

Hypothesis C:

```text
Even if support-seeking regulation is observed, identity acceptance, semantic
goal ownership, intention, agency, and fully native integration remain blocked
unless separate theory and native implementation gates are satisfied.
```
