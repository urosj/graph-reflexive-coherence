# Hypothesis A - Boundary Source Inventory

## Claim

Internal support-relevant state and external resource or perturbation state can
be inventoried as distinct, source-backed artifact surfaces without accepting
externally supplied boundary labels.

The inventory must support a two-axis stable basin-boundary requirements
matrix, not a complete self/environment taxonomy.

## Required Evidence

- Source-backed rows for internal support-relevant state.
- Source-backed rows for external resource or perturbation state, or explicit
  absence records where no such source exists.
- Source-backed rows for external structured state where C3 false-positive
  pressure is used.
- Boundary-side labels derived from pinned artifacts, not supplied by the
  experiment after the fact.
- Boundary-state axis rows for B0-B4, each with lineage sources, inherited
  closed claims, constructed support, unsupported extension, required N16
  evidence, and claim ceiling.
- Challenge-class axis rows for C0-C5, recorded as N16 operational test
  conditions rather than inherited environment taxonomy.
- Selected interaction cells for B0 x C3, B1 x C2, B2 x C1, B3 x C4, and
  B4 x C5.
- Explicit classification that C3 structured external coherence is not
  perturbation unless crossing or disruption is recorded.
- SHA-256 digests for every consumed source artifact and report.
- Explicit claim ceilings for every source row.

## Controls

- Externally supplied boundary label blocked.
- Hidden external-state injection blocked.
- Missing boundary-side source blocked.
- Stale internal or external state blocked.

## Non-Claims

This hypothesis does not claim selfhood, personhood, identity acceptance,
semantic goal ownership, agency, native support, or fully native integration.
