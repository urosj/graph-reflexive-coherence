# N16 - LGRC Self / Environment Boundary

N16 is an agency-prerequisite experiment. It starts after the N15 endogenous
proxy formation closeout and asks whether internal support-relevant state can
be separated from external resource or perturbation state in artifacts and
controls.

The core question is:

```text
Can internal support-relevant state and external resource or perturbation state
be separated in artifacts and controls without promoting the boundary into
selfhood, identity acceptance, agency, or native support?
```

N16 does not prove selfhood. It does not prove personhood, biological behavior,
identity acceptance, semantic goal ownership, intention, semantic choice,
agency, unrestricted agency, native support, or fully native agentic-like
integration.

## Scope

N16 should not try to prove every kind of self/environment boundary. N16 is a
requirements-discovery experiment for stable basin boundaries across two axes:
lineage-derived boundary state and operational challenge class.

The guiding question is:

```text
What does an LGRC basin need in order to remain a basin when it is exposed to
boundary challenges?
```

N16 scope is:

```text
basin-boundary stability requirements across boundary-state and
challenge-class interaction cells
```

N16 does not cover:

```text
selective uptake
resource assimilation
porous boundary function
agency-like boundary behavior
semantic recognition
organism or life claims
full failure taxonomy
```

Those move from boundary stability into boundary-mediated function and should
be postponed to later work.

## Evidence Strategy

N16 should use direct historic evidence if the inventory finds source-backed,
claim-clean records that already separate internal support-relevant state from
external resource or perturbation state. Such evidence must still pass
freshness, budget, replay, and claim-boundary controls.

The strongest expected proof path is a constructed AP6 candidate assembled from
old best closed claims:

```text
N15 AP5 endogenous target/proxy formation
N14 AP4 consequence-sensitive route selection
N13 AP3 support-seeking regulation
N08 memory context
N09 bounded regulation context
N12 NAT4 readiness records as readiness-only context
```

This construction must preserve each input's original claim ceiling. N15 does
not become goal ownership, N14 does not become intention, N13 does not become
selfhood, and N12 does not become native support.

The AP5 to AP6 bridge is:

```text
AP5:
    target/proxy condition is generated before downstream use from
    source-current runtime-visible artifact state

AP6:
    internal support-relevant state and external resource/perturbation state
    are separately represented, traceable, and controlled
```

N16 is not a relabel of AP5. The additional capability is a boundary record
that keeps internal support-relevant state and external state separable under
controls, replay, and challenge pressure.

## Two-Axis Matrix

N16 uses a bounded requirements matrix. The first axis is boundary state. This
axis must be derived from prior N** evidence, not invented as a new generic
border taxonomy. Iteration 1 must record source-backed lineage for every B
state using these fields:

```text
lineage_sources
inherited_closed_claims
constructed_support
unsupported_extension
required_N16_evidence
claim_ceiling
```

The provisional B-axis is:

```text
B0 null / external coherence only
    Active null. Prior controls can support rejection of relabeled external
    structure, but no boundary claim is allowed.

B1 localized basin partition
    Basin-like inside/outside partition and boundary edges can be extracted.
    This draws mainly on N03 basin loops and N04 boundary-coupling taxonomy.

B2 support-persistent basin
    Boundary remains stable across time/replay. This draws mainly on N07
    identity/support persistence and later AP support/replay records.

B3 regulated repair / reabsorption boundary
    Boundary can reabsorb leakage or recover after disruption. This draws on
    N09 bounded recovery, N13 support response, and N15 drift/replay controls.

B4 coupled multi-basin separability candidate
    Separate basins remain distinct while coupled through shared substrate.
    This is the weakest inherited state: N08/N14/N15 supply partial analogs,
    but N16 must produce the separability evidence.
```

The second axis is not an inherited environment taxonomy. It is an operational
challenge-class axis: the minimum challenge basis needed to test whether a
candidate boundary persists, rejects false positives, repairs, or remains
separable.

```text
C0 quiet reference
    Calibration condition with no intended boundary stress.

C1 unstructured perturbation
    Random or noisy disturbance around or across the boundary.

C2 directional flux
    One-sided pressure, drift, or flow across the boundary.

C3 structured external coherence
    Active null: coherent outside pattern that must not be mistaken for the
    basin boundary or self-region. This is external structured state that
    creates false-positive pressure, not perturbation by default. It becomes
    perturbation only if a crossing or disruption is explicitly recorded.

C4 breach and repair
    Local boundary disruption followed by a test for reclosure, reabsorption,
    or fail-closed classification.

C5 coupled neighbor / shared medium
    More than one candidate basin interacting through a shared substrate.
```

N16 should not run the full Cartesian product blindly. It should freeze both
axes, then run main-effect sweeps plus selected interaction probes:

```text
B0 x C3
    structured external coherence active null; must reject false boundary

B1 x C2
    weak detectable boundary under flux; likely failure threshold

B2 x C1
    persistent boundary under unstructured perturbation

B3 x C4
    breach and repair / reclosure probe

B4 x C5
    multi-basin exclusivity in shared medium
```

B3 repair/reabsorption cells are not unlocked until B2 has been evaluated
under at least C0, C1, and C2, or until those cells produce explicit blockers.
Repair must not hide a failure to establish the basic support-persistent
boundary.

B4 may appear in Iteration 5 as a C2 flux stress row, but its primary target
is B4 x C5 in Iteration 6. If the multi-basin substrate is not sufficiently
source-backed, the Iteration 5 B4 row must be `partial` or `not_applicable`
rather than promoted as separability evidence.

## Hypotheses

Hypothesis A:

```text
Internal support-relevant state and external resource or perturbation state can
be inventoried as distinct, source-backed artifact surfaces without accepting
externally supplied boundary labels.
```

Hypothesis B:

```text
An artifact-level basin-boundary candidate can remain distinguishable across
selected boundary-state/challenge interaction cells under bounded drift,
freshness, budget, dependency-trace, and replay constraints.
```

Hypothesis C:

```text
Even if internal/external separability is observed, selfhood, personhood,
identity acceptance, semantic goal ownership, intention, agency, native support,
fully native integration, and unrestricted agency remain blocked unless
separate theory and native implementation gates are satisfied.
```
