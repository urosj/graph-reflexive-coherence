# N31 Hypotheses

N31 tests six linked hypotheses:

```text
A = coherence-only derived causal decay
B = observable-versus-causal distinction
C = added-mechanism semantic discrimination
D = conservation, internal time, and local encounter
E = representation, restoration, and reconstruction
F = claim boundary and RCAE return integrity
```

Hypotheses A, B, and C are alternative or comparative scientific outcomes.
They may support an N31 result only if D, E, and F remain satisfied. A failure
of D, E, or F blocks stronger decay-relation claims regardless of local effect
size.

Across all hypotheses, route mass, route organization, and causal mediation are
independent evidence contracts. No one contract may substitute for either of
the others. D0-R and B-R refine D0 and Candidate B respectively; they do not
add a seventh primary candidate class.
