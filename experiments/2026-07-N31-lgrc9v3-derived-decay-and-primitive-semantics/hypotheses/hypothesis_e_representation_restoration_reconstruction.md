# Hypothesis E - Representation, Restoration, And Reconstruction

## Statement

N31 can determine whether each claimed carrier is represented adequately and
can restore, branch, replay, and reconstruct every causally load-bearing state
without substituting coarse equality or report synthesis.

## Required Support

- D0a representation status is frozen before positive execution.
- Route mass, route organization, and causal mediation each have a declared
  representation/authority contract before positive execution.
- Exact projections freeze basis, decomposition/recomposition, overlap policy,
  temporal support, intervention semantics, and error bound.
- Lossy or missing representation blocks D0a rather than triggering invented
  state.
- Reset-sensitive rows use LGRC9V3 restoration identity v2.
- External producer/experiment state is serialized and composed separately.
- Equal-state branches continue equivalently under declared inputs.
- Derived-cache recomputation and complete execution reconstruction are
  reported separately.
- Future builders and validators reject rows using a superseded schema unless
  they migrate deterministically or are regenerated.

## Failure Conditions

- Node scalar equality stands in for complete route-local state equality.
- Restoration identity v1 is treated as reset-aware.
- Hidden producer state is omitted from snapshots or branches.
- A report digest substitutes for runtime reconstruction.
- A cache changes continuation after removal and exact recomputation.
- A lossy scalar, cached diagnostic, or producer-owned state is used as ordinary
  coherence-only route organization.

## Claim Effect

A missing representation is a valid N31 result. It is not evidence against the
coherence-only theory and cannot be bypassed to earn a positive D0a row.
