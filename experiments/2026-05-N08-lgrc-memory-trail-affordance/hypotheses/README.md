# N08 Hypotheses

N08 keeps two route-memory hypotheses separate so evidence from one branch does
not accidentally promote the other.

```text
Hypothesis A:
    serialized producer/policy memory

Hypothesis B:
    native geometry-mediated trail memory
```

Current status:

```text
Hypothesis A:
    closed through Iteration 8 at MEM6
    ceiling = artifact_only_route_memory_or_trail_affordance_candidate
    result = ACO-like producer design pattern for serialized route memory

Hypothesis B:
    closed through Iteration 13 with bounded native-policy-gap result
    ceiling = static_positive_geometry_route_response_persistence_candidate
    blocker = native_route_conductance_memory_policy_missing
```

The narrow Iteration 8 memory/trail claim applies only to Hypothesis A. It does
not imply native geometry-mediated memory, pure coherence/flux trail memory,
ACO, ant-colony behavior, agency, intention, goal regulation, locomotion,
biological pheromone behavior, identity acceptance, or unrestricted movement.

The Iterations 9-12 Hypothesis B branch is aligned with the N05-N11 roadmap as
a producer/scaffold-to-native-policy discovery result: it uses declared
runtime-visible geometry and native route arbitration without independent
`memory_strength`, but it does not yet show LGRC-native conductance update,
strengthening, relaxation, or pure flux trail memory. That missing native
policy remains the useful blocker for later Phase 8/native absorption.
