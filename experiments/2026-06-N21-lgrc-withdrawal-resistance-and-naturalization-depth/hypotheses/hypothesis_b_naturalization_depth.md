# Hypothesis B - Naturalization Depth After Probe Or Scaffold Removal

N21 can construct source-backed evidence that a declared basin signature
continues after the original probe or scaffold is absent, without treating
producer residue as native support.

This hypothesis supports bounded candidate evidence on the N21-local ND ladder.
Probe-absent persistence is not by itself a general naturalization-depth
classification.

## Expected Support

Hypothesis B is supported only if a naturalization-depth row records all of:

```text

probe residue control fails closed
support source annotation relabel control fails closed
hidden support control fails closed

unsafe claim flags false
Hypothesis C passes
```

## Failure Conditions

Hypothesis B fails if any positive row depends on:

```text

hidden producer support
proxy improvement without post-probe basin continuation

probe residue digest present

native absorption by label
selfhood or identity relabel
sentience or consciousness relabel
Hypothesis C fails
```
