# Hypothesis A - Source-Current Transfer Mapping

N27 can produce source-current evidence that a declared configuration,
topology, fixture, or substrate mapping exists before use and yields an
auditable post-transfer basin signature.

Expected support requires:

```text
N20 configuration/substrate transfer contract consumed
mapping declared before use
mapping digest recorded
mapping source-backed
pre-transfer basin signature trace present
post-transfer basin signature trace present
boundary mapping trace present
support preservation trace present
coherence preservation trace present
flux balance trace present
artifact manifest present
artifact hashes match file contents
derived_report_only = false
```

Failure conditions:

```text
mapping declared after outcome inspection
fixture equivalence label without mapping trace
same label assigned to different basin
post-transfer signature missing
boundary mapping missing
support/coherence floor missing
hidden support reconstruction supplies the apparent transfer
proxy score substitutes for basin signature preservation
cross-substrate transfer claimed without declared source-backed substrate mapping
```

Hypothesis A cannot by itself support semantic identity, agency, native support,
sentience, Phase 8 completion, or ant ecology.
