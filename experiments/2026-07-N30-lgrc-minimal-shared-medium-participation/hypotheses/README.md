# N30 Hypotheses

N30 tests four hypotheses:

```text
A = shared-medium source-basis and vocabulary admission
B = minimal participant continuity and attribution
C = medium surface perturbation / trace / eligibility evidence
D = medium debt, controls, and claim-boundary preservation
```

Hypotheses B and C may support N30 only if Hypothesis D also passes. If
direct-message, global-controller, hidden-producer, label-only, or semantic
relabel paths remain open, positive-looking medium relations are demoted or
blocked.
