# N30 Hypotheses

```text
A = shared-medium source-basis and vocabulary admission
B = minimal participant continuity and attribution
C = medium surface perturbation / trace / eligibility evidence
D = medium debt, controls, and claim-boundary preservation
E = coupled shared-medium relation chain
```

Hypotheses B and C may support N30 only if Hypotheses D and E also pass. If
direct-message, global-controller, hidden-producer, label-only, private-state,
post-hoc chain, or semantic relabel paths remain open, positive-looking medium
relations are demoted or blocked.
