# N25.1 - LGRC9V3 Multi-Basin Formation Extension Requirements

N25.1 is a requirements/specification bridge after N25 and before N26.

N25 closed with scoped native BF5 / N25-C6 evidence:

```text
scoped high-margin core / sub-basin formation supported
producer-assisted BF5 scaffold context supported
independent new-basin formation blocked
native LGRC9V3 multi-basin formation blocked
BF6 blocked
```

The important correction is that GRC-9/GRC9V3 port-9 spark mechanics are
mechanical refinement first. They can create substrate room for child basins,
but they do not directly impose multiple basins. LGRC9V3 currently wraps and
causalizes spark/topology/refinement surfaces, but N25 did not establish a
native LGRC9V3 mechanism that forms multiple stable basins.

N25.1 defines the missing Phase 8 extension contract:

```text
causal spark or boundary-birth event
-> topology integration / mechanical refinement
-> post-refinement flow window
-> child-basin core extraction
-> child-basin support/coherence records
-> merge/leakage and relabel controls
-> replayable child-basin persistence
```

This package is not a runtime implementation. It must not claim:

```text
native multi-basin formation
independent new-basin formation
BF6
native support
semantic learning
semantic choice
agency
sentience
Phase 8 implementation complete
ant ecology
```
