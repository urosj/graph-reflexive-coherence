basin-forming capacity of its environment or neighborhood, without reducing the
result to focal survival, transfer success, merge/leakage, semantic

Core question:

```text
Can one basin persist while increasing the basin-forming capacity of its
environment or neighborhood?
```

LGRC reading:

```text
generative persistence =
  focal basin remains stable while neighboring or sub-basin structures become
  more distinguishable, better supported, or more capable of preserving their
  own boundaries

extractive persistence =
  focal basin remains stable by flattening, draining, merging, or suppressing
  neighboring basin-forming capacity
```

N28 is the first N20-N29 experiment that directly touches ecology-like
structure. It remains inside this repository and must not specify ant ecology
behavior, colony agency, semantic cooperation, organism/life, or native
support. N29 remains the first formal bridge into the agentic ecology project.
