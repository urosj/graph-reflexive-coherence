# N17 - LGRC Closed Boundary Engagement Loop

N17 is an agency-prerequisite experiment. It starts after the N16
self/environment boundary closeout and asks whether an artifact-level AP6
boundary candidate can support a source-current closed boundary engagement
loop.

The core question is:

```text
Can an artifact-level AP6 boundary candidate support a closed environmental
engagement loop in which external state changes internal support, internal
response changes external state, and the changed external state feeds back into
later internal support?
```

N17 does not prove agency. It does not prove selfhood, personhood, biological
behavior, identity acceptance, semantic goal ownership, intention, semantic
choice, native support, fully native agentic-like integration, organism/life,
or unrestricted agency.

## Scope

N17 is the first closed-loop experiment after boundary. N16 separated inside,
outside, and crossing. N17 tests whether crossing becomes a source-current
loop.

```text
N16:
    Can we distinguish inside, outside, and crossing?

N17:
    Can crossing become a source-current loop?
```

N17 scope is:

```text
artifact-level closed boundary engagement loop candidate
```

The ordered trace shape is:

```text
external state -> boundary/internal support change
internal support response -> external state change
changed external state -> later internal support change
```

The required monotonic phase ordering is:

```text
t0 external pressure/crossing
t1 internal support update
t2 response-caused external change
t3 later internal support conditioned by changed external state
```

N17 must not prove "action" by naming an outbound change action. It must prove
closure by showing ordered trace dependence:

```text
external -> internal -> external -> later internal
```

N17 does not cover:

```text
semantic perception
semantic action
intention
choice
goal ownership
agency
selfhood
organism/life
native support
fully native integration
unrestricted autonomous loop
```

## Evidence Strategy

N17 should use direct historic evidence if the inventory finds source-backed,
claim-clean records that already show closed loop ordering:

```text
external state changes internal support
internal response changes external state
changed external state changes later internal support
```

If direct support exists, it must still pass freshness, replay, order,
negative-control, and claim-boundary checks.

The expected proof path starts from N16's AP6 boundary and asks whether the
boundary-crossing trace becomes closed:

```text

N15 AP5 -> source-current target/proxy context
```

The construction must preserve every source experiment's claim ceiling. N16
does not become agency, N15 does not become goal ownership, N14 does not
become intention, N13 does not become selfhood, and N12 does not become native
support.

## Loop Ladder

N17 should use a local loop ladder:

```text
N17-G0:
    one-way boundary crossing trace

N17-G1:
    internal update after external crossing

N17-G2:
    outbound response changes external state

N17-G3:
    changed external state feeds back into later internal state

N17-G4:
    replay-stable closed loop

N17-G5:
    challenge-stable closed loop under perturbation or flux

N17-G6:
    shared-medium closed loop without basin merge

N17-G7:
    claim-clean AP7 candidate, unsafe promotions blocked
```

The key rung is `N17-G3`. That is where a crossing trace becomes a closed
loop candidate.

Claim-boundary classification is tracked separately from evidence-rung
advancement. Iteration 6 may classify the replay/control-clean MVP row as an
artifact-level AP7 MVP candidate, but the current evidence rung remains G4
until Iteration 6-A or 6-B tests G5 challenge stability.

Formal admissibility rule:

```text
G3 is the first admissible closed-loop evidence rung.
Rows below G3 may be useful, but they cannot support AP7.
G0/G1/G2 are near-misses, not closed-loop evidence.
```

## Example Families

N17 should start with a minimal closed loop before the shared-medium case.

```text
1. Perturbation-response-recovery loop
   external perturbation crosses boundary
   internal support shifts
   bounded response or reclosure occurs
   external perturbation field changes
   later internal support depends on changed external state

2. Resource/support modulation loop
   external resource/support condition changes
   internal support-relevant state updates
   response changes access/path/pressure around the resource
   modified resource condition changes later internal support

3. Shared-medium reciprocal loop
   basin A and basin B share a medium
   shared-medium condition changes A
   A response changes shared medium
   changed shared medium affects B or later A
   separability remains preserved
```
