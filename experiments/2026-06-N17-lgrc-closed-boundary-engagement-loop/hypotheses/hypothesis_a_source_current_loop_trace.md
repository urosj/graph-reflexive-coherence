# Hypothesis A - Source-Current Loop Trace

## Claim

A source-current trace can show external challenge or resource state affecting
internal support-relevant state, followed by an internal response that modifies
external state, followed by a later internal update conditioned by that
modified external state.

## Required Evidence

```text
external_to_internal_trace
internal_response_trace
response_to_external_change_trace
external_feedback_to_internal_trace
loop_closure_evidence
```

## Boundary

This supports only an artifact-level loop trace candidate. It does not support
agency, intention, semantic perception, semantic action, goal ownership,
selfhood, native support, or organism/life claims.
