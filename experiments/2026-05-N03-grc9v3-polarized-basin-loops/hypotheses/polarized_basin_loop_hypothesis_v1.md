# Polarized Basin Loop Hypothesis

## Primary Hypothesis

A fixed-topology GRC9V3 basin can exhibit a closed internal redistributive loop
in which one candidate aspect region becomes export-biased, another becomes
import-biased, and a return channel later refills the export-biased region,
all under conserved total coherence.

The first positive target is:

```text
L4 = repeated conserved two-aspect cycle
```

## Null Expectations

The null lanes should not produce strong loop evidence:

```text
U0 uniform field -> no persistent source/sink polarity
U1 symmetric bump -> structure without directed loop polarity
U2 closed substrate/masks -> closed path without measured loop roles
```

## Claim Boundaries

The first implementation can claim at most a two-aspect polarized basin-loop
result. It must not claim:

```text
movement
locomotion
agency
identity fission
external source/sink dynamics
multi-pole basin-loop evidence
```

Because the first fixture uses:

```text
same_parent_basin_mode = configured_parent_region_only
```

early positive runs remain candidate loop claims unless stronger
`flux_successor_basin` evidence is implemented and audited later.

## Required Evidence For L4

An L4 candidate requires:

```text
fixed topology
budget conservation
measured source/sink roles
return-channel evidence
n_cycles_min >= 3
post-washout evaluation
null separation
ordered phase-cascade evidence
```

The ordered phase cascade is:

```text
C_source drop
-> J_forward rise
-> C_sink rise
-> J_return rise
-> C_source refill
```

## Falsification / Downgrade Conditions

Downgrade or block positive claims if:

```text
topology changes
budget correction dominates the signal
source/sink roles depend on labels rather than measured flux
U0/U1/U2 nulls produce comparable evidence
randomized labels still produce loop classification
phase cascade is unavailable or not ordered
same-parent evidence is unavailable
```

## Deferred Surfaces

The multi-pole Appendix A family is deferred until the minimal two-aspect L4
result is stable and reproducible.
