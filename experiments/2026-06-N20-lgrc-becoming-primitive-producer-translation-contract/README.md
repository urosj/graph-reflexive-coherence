# N20 - LGRC Becoming-Primitive Producer Translation Contract

N20 closes the N20-N29 becoming-agency ecology roadmap entry point as a
contract experiment. It does not test withdrawal, learning, choice, abundance,
spark, proxy collapse, transfer, generative persistence, or ecology.

The core question is:

```text
Which agency-of-becoming diagnostics can be expressed as LGRC-visible geometry,
and which still require explicit producer surfaces?
```

N20 is deliberately prior to primitive evidence. Its completed job is to define
the language, schemas, claim boundaries, and producer-residue accounting that
later experiments must obey.

## Producer Definition

A producer is an explicit implementation surface that introduces, updates,
routes, schedules, labels, or preserves a quantity that is not yet carried by
LGRC source-current geometry.

A producer is not the agent. It is the current implementation residue left
after trying to express apparent agency state as basin, boundary, support,
flux, coupling, aftereffect, susceptibility, trace, reserve, or persistence
condition.

Every variable in the N20 contract must be classified as one of:

```text
substrate_carried
producer_mediated
naturalization_debt
blocked_relabel
```

For `naturalization_debt`, records must also classify the debt subtype:

```text
telemetry_debt
policy_debt
state_mutation_debt
replay_debt
budget_debt
source_currentness_debt
claim_boundary_debt
```
