# Hypothesis A - Becoming Diagnostic Translation

The agency-of-becoming diagnostics can be translated into LGRC-visible
primitive contracts without promoting interpretive labels into primitive
implementation state.

Expected support:

```text
each diagnostic has an LGRC-visible field mapping
each diagnostic has a blocked semantic overclaim list
each diagnostic names its producer-mediated fields
each diagnostic names its naturalization debt
each diagnostic declares what counts as same-basin continuation
each diagnostic declares what counts as proxy-only success
each diagnostic has a handoff target in N21-N28
```

Failure condition:

```text
agency, choice, learning, function, goal, intention, sentience, or identity
labels are used as primitive evidence instead of bounded interpretation
```
