The experiment family keeps the original semantic hypotheses as the phenomena
under investigation, and adds discriminator hypotheses that define how those
phenomena can be falsified.

H0: Anonymous-port null
    Rows, columns, and ports add no explanatory power beyond ordinary graph
    degree, adjacency, conductance, coherence, and edge labels.

O1: Row semantic hypothesis
    Rows behave as local differential modes.

O2: Column semantic hypothesis
    Columns behave as interface, routing, refinement, and coarse-graining
    families.

O3: Port intersection hypothesis
    Ports carry row-column interaction behavior not reducible to row-only or
    column-only grouping.

O4: Edge-label path hypothesis
    Metric, temporal-delay, and strongest-flux paths can disagree while
    remaining auditable through existing edge labels.

D1: Factorization discriminator
    Behavior is equivariant under S3 row and S3 column permutations, but not
    under arbitrary S9 port relabeling.

D2: Predictive separation discriminator
    Rows predict geometric/differential artifacts better than columns.
    Columns predict interface/refinement artifacts better than rows.
    Ports predict edge-local artifacts better than either alone.

D3: Transpose discriminator
    Row/column transpose is not behaviorally equivalent.

D4: Saturation discriminator
    Nine-port fullness functions as representational bottleneck / chart
    exhaustion under canonical spark rules.

D5: Interface-memory discriminator
    Parent column labels remain predictive across refinement.

D6: Port-interaction discriminator
    Some responses require row × column interaction terms.

D7: Multiscale discriminator
    Column G/Split reconstructs eligible nonnegative port fields, while exact
    signed flux reconstruction requires J+ / J- decomposition.

D8: Identity-emergence discriminator
    Mechanical refinement is not identity fission unless persistent child
    sink/basin artifacts appear.
