# Implementation Trace

This directory records what was done for the
`2026-05-01-grc9v3-properties` experiment family and why.

Use this area for experiment-local notes such as:

- run plans,
- fixture and seed design notes,
- handoff notes,
- observed failures,
- interpretation changes,
- report closeouts,
- and links to generated artifacts.

This is not the repo-level `implementation/` directory. It should not describe
project-wide runtime contracts unless an experiment result is being promoted
into a separate repo-level implementation task.
