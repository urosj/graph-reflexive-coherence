# N18 Long-Horizon Agentic-Like Closure Stress Test Implementation Plan

## Purpose

N18 tests whether the N17 artifact-level AP7 closed boundary engagement loop
and its supporting N12-N16 agency-prerequisite stack remain source-current,
replay-clean, budget-clean, and claim-clean over longer horizons and
perturbations.

The thesis is:

```text
N18 tests whether identity/support, memory, regulation,
consequence-sensitive selection, proxy formation, boundary separation, and
closed-loop feedback remain linked over longer horizon windows and stress
families without stale state, hidden native support, post-hoc stitching, or
unsafe agency relabels.
```

The bounded conclusion, if supported without Phase 8 implementation, is:

```text
artifact_only_agentic_like_closure_candidate
```

Internally this maps to:

```text
AP8 = long-horizon agentic-like closure candidate
```

N18 must not support agency, choice, intention, semantic perception, semantic
action, semantic goal ownership, native support, selfhood, identity acceptance,
life, organism behavior, fully native integration, or unrestricted agency.

## Method

N18 should continue the Arc of Becoming method used in the N12-N18 tranche:

```text
inventory old-best evidence
freeze schema and gates before positive interpretation
construct only from source-current old-best claims
stress the constructed result under explicit controls
classify claims with unsafe promotions blocked
close with a handoff rather than silent native implementation
```
