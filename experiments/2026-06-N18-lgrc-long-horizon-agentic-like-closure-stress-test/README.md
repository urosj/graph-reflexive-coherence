# N18 - LGRC Long-Horizon Agentic-Like Closure Stress Test

N18 is an agency-prerequisite experiment. It starts after the N17
artifact-level AP7 closeout and asks whether the closed boundary engagement
loop can remain source-current, replay-clean, budget-clean, and claim-clean
over longer horizons and perturbation families.

The core question is:

```text
Can identity/support, memory, regulation, consequence-sensitive selection,
proxy formation, boundary separation, and closed-loop feedback remain
source-current, replay-clean, budget-clean, and claim-clean under longer
horizons and perturbations?
```

N18 does not prove agency. It does not prove intention, semantic action,
semantic perception, semantic goal ownership, selfhood, personhood, biological
behavior, identity acceptance, native support, fully native agentic-like
integration, organism/life, or unrestricted agency.

## Scope

N18 stress-tests the AP7 closed boundary engagement loop over longer windows
and across multiple source-backed agency-prerequisite pillars:

```text
support / identity-relevant stability
memory / route-context history
bounded regulation
consequence-sensitive route selection
endogenous proxy formation
self/environment boundary separation
closed boundary engagement feedback
```

The maximum artifact-level target is:

```text
artifact_only_agentic_like_closure_candidate
```

Internally this maps to the roadmap's AP8 target:

```text
AP8:
    long-horizon agentic-like closure candidate
```

AP8 is not agency. It is the strongest agency-prerequisite evidence target for
the N12-N18 tranche. A stronger native result may be considered only if a
separate Phase 8 implementation has actually implemented and validated native
support.

## Evidence Strategy

N18 should use direct historic evidence when the inventory finds source-backed,
claim-clean long-horizon records that already cover the AP8 stress target. If
direct support exists, it must still pass freshness, replay, order, budget,
negative-control, and claim-boundary checks.

The expected construction starts from the old-best closed claims:

```text
N17 AP7 -> artifact-level closed boundary engagement loop
N16 AP6 -> controlled self/environment boundary separation
N15 AP5 -> endogenous proxy formation context
N14 AP4 -> consequence-sensitive selection context
N13 AP3 -> support-seeking regulation context
N12 NAT4 -> readiness-only native absorption context
N08/N09 -> memory and bounded regulation context when source-pinned
```

N18 should not construct AP8 by relabeling any one prior result. It should test
whether the composed agency-prerequisite stack remains stable under longer
horizon windows, route/context reversals, support withdrawal and restoration,
proxy perturbation, memory relaxation, environment/resource perturbation,
shared-medium perturbation, duplicate/replay controls, and artifact-only
reconstruction.

AP8 support is horizon-bounded. A supported row can support only the explicit
tested horizon window and stress envelope recorded in that row; horizon
extrapolation is not allowed.
