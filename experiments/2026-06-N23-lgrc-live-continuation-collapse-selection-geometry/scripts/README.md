# N23 Scripts

Reconstruction scripts for N23 belong here.

Scripts should generate deterministic artifacts, declare thresholds before
outcomes are inspected, write repository-relative paths only, and keep branch
collapse evidence separate from producer labels, semantic choice terminology,
and N22 susceptibility-update context.

Current visualization support:

```text
render_n23_selection_collapse_visualization.py
```

This renderer creates supporting artifact-level graph/sequence/animation
visuals from the N23 four-branch live-continuation collapse row. The visuals
are inspection aids only; they do not add semantic choice, agency, native
support, sentience, Phase 8, or ant-ecology evidence.
