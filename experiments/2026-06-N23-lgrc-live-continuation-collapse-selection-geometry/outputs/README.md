# N23 Outputs

Generated JSON artifacts and supporting visual assets for N23 belong here.

Outputs should be deterministic, replayable, and free of local absolute paths.
They must distinguish source-current live branch/collapse geometry from
producer-selected labels, producer preferences, random ties, proxy metrics, and
semantic choice terminology.

Supporting visualization artifacts:

```text
n23_selection_collapse_visualization.json
n23_selection_collapse_visualization/
```

The visualization manifest records source artifact hashes and keeps the visual
status as supporting artifact-level inspection only.
