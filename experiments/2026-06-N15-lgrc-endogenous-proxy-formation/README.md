# N15 - LGRC Endogenous Proxy Formation

N15 is an agency-prerequisite experiment. It starts after the N14
consequence-sensitive route selection closeout and asks whether proxy or target
conditions can be generated from runtime-visible support, memory, or regulation
state rather than being supplied by the experiment.

The core question is:

```text
Can proxy or target conditions arise from runtime-visible support, memory, or
regulation state rather than being declared externally, while preserving
claim boundaries, replay, and budget controls?
```

N15 does not prove semantic goal ownership. It does not prove intention,
semantic choice, agency, identity acceptance, selfhood, personhood, biological
behavior, native support, or fully native agentic-like integration.

## Evidence Strategy

N15 should use direct historic evidence if the inventory finds source-backed,
claim-clean records that already support runtime-derived proxy or target
formation. Such evidence must still pass freshness, budget, replay, and claim
boundary controls.

The strongest proof path is not a direct relabel of an old row. It is a
constructed AP5 candidate assembled from the old best closed claims:

```text
N13 AP3 support-seeking regulation
N14 AP4 consequence-sensitive route selection
N08 route memory context
N09 bounded regulation context
N12 NAT4 readiness records as readiness-only context
```

This construction must preserve each input's original claim ceiling. N13 does
not become selfhood, N14 does not become intention or goal ownership, and N12
does not become native support.

The old-best-claims construction method is trace-preserving composition:

```text
1. consume N13 AP3 only as the support-seeking regulation axis
2. consume N14 AP4 only as the consequence-sensitive selection axis
3. consume N08 only as the memory/context axis
4. consume N09 only as the bounded-regulation axis
5. consume N12 NAT4 only as readiness-only context
6. derive a target condition from serialized source-current inputs
7. record a field-level dependency trace for every derived target field
8. assign no source input a claim stronger than its closed claim ceiling
```

The AP4 to AP5 bridge is:

```text
AP4:
    selection uses source-backed downstream consequence records

AP5:
    a target/proxy condition is generated before downstream use from
    source-current runtime-visible artifact state
```

N15 uses AP4 as the strongest available construction substrate, but AP5 is not
a relabel of AP4. The additional capability is pre-use target formation from
runtime-visible artifact state.

## Hypotheses

Hypothesis A:

```text
Runtime-visible support, memory, regulation, or support/identity-condition
state can be inventoried as a bounded source surface for proxy formation
without accepting externally supplied target conditions.
```

Hypothesis B:

```text

source-current runtime-visible state under bounded drift, freshness, budget,
and replay constraints rather than from declared target fixtures.
```

Hypothesis C:

```text
Even if endogenous proxy formation is observed, semantic goal ownership,
intention, semantic choice, agency, identity acceptance, selfhood, personhood,
biological behavior, native support, fully native integration, and unrestricted
agency remain blocked unless separate theory and native implementation gates
are satisfied.
```
