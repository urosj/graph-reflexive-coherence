# N15 Hypotheses

N15 hypotheses are specific to endogenous proxy formation. They are not a
generic continuation of N14 consequence-sensitive route selection.

The experiment asks whether proxy or target conditions can be generated from
runtime-visible support, memory, regulation, or support/identity-condition
state while blocking externally injected targets, hidden derivations, post-hoc
formation, unbounded drift, goal-ownership relabels, identity-acceptance
relabels, agency relabels, and native-support relabels.

- `hypothesis_a_runtime_state_proxy_sources.md`
- `hypothesis_b_bounded_endogenous_proxy_formation.md`
- `hypothesis_c_goal_ownership_and_agency_boundary.md`
