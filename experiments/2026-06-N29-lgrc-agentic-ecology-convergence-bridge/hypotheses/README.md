# N29 Hypotheses

N29 tests five bridge hypotheses:

```text
A = ecology demand and capability supply grounding
B = coverage/debt matrix and bridge motif correctness
C = prototype atlas admissibility
D = composition and first-probe handoff
E = claim boundary and native-readiness preservation
```

Hypotheses A-D may support N29 only if Hypothesis E also passes. If the claim
boundary fails, positive-looking mappings or prototypes are demoted or blocked.
