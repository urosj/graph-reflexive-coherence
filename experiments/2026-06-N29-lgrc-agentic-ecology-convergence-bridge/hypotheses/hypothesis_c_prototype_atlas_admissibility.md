# Hypothesis C - Prototype Atlas Admissibility

N29 should produce bridge exemplars wherever the source record permits it, not
only a paperwork mapping. A prototype is admissible only if it shows how a
source-backed capability can serve an ecology demand and what downstream probe
should test the composition.

Expected support:

```text
prototype admission schema is frozen before prototype rows
each prototype has source artifacts or a declared reconstruction basis
each prototype records runtime_or_reconstruction_status
each prototype maps to ecology demand, supplied capability, bridge motif, and
agency diagnostic roles
each prototype states what exact next ecology probe it suggests
each prototype records producer residue and medium debt
each prototype keeps unsafe claim flags false
mapping-only rows explain why no prototype is available
```

Prototype classes:

```text
trace / aftereffect
reserve / pressure
boundary / mobile expression
closed-loop perturbation
proxy collapse
transfer
generative/extractive medium reshaping
composition
```

Failure conditions:

```text
prototype row lacks source basis
prototype lacks downstream probe implication
visual or report-only artifact is treated as proof
prototype relabels itself as ant behavior
prototype relabels producer scaffold as native ecology
composition is asserted without controls
```
