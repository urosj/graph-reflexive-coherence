# N29 - LGRC Agentic Ecology Convergence Bridge

N29 is the convergence bridge between the full `N05-N28` experiment record,
the agency-of-becoming diagnostic interpretation, and the
`reflexive-coherence-agentic-ecology` specification repository.

The center of gravity is not re-validating the previous experiments. It is:

```text
What does agentic ecology need?
What can N05-N28 actually supply?
What bridge compositions can be built now?
What remains producer / medium / naturalization debt?
What runnable ecology probes become possible?
```

N29 should therefore produce bridge artifacts and small source-backed
structures, not another long internal proof ladder.

Repository role:

```text
this repository:
  core LGRC/GRC runtime
  essential primitive building blocks
  minimal producers that expose missing native mechanisms
  source-backed experiments proving specific properties
  bridge artifacts showing which combinations are safe to try next

agentic ecology repository:
  composition space
  domain models
  multi-component dynamics
  ecology-specific probes
  higher-order behavior from combinations of LGRC/GRC primitives
```

N29 is the transition layer between those roles. It should show which
combinations are coherent enough to carry into the ecology repository, while
also preserving the debt and claim limits that prevent overclaiming.

## What N29 Is Not

N29 is not:


N29 can reuse existing artifacts, closeouts, visualizations, and runtime
examples. It only rechecks enough source status to prevent overclaiming.

## Bridge Motifs

N29 should define motifs that connect ecology demands to available LGRC/GRC
capabilities. A motif is a candidate structure that can become a prototype or a
probe contract.

Initial motif families:

```text
trace -> pressure -> loop:
  route/support aftereffects become pressure surfaces for later loop response

reserve -> optionality -> basin formation:
  surplus above maintenance floor enables optional continuation or formation

boundary -> shared medium -> multi-basin unit:
  boundary and child-basin evidence becomes a bounded shared-medium unit

proxy collapse -> susceptibility -> re-entry:
  proxy divergence changes routing pressure and later re-entry behavior

transfer -> replay -> role relocation:
  configuration transfer carries basin-relevant structure across substrate frames

generative/extractive medium reshaping:
  persistence pattern changes neighboring basin-forming capacity

composition:
  two or more motifs are coupled with explicit order, controls, and debt
```

Motifs are bridge structures. They are not native ecology proof unless a future
runtime probe supplies that evidence.

## Prototype Atlas Policy

N29 should provide small running examples, source-backed reconstructions, or
diagnostic visual structures wherever possible.

```text
Every mapping row should prefer a runnable/source-backed miniature.
If no miniature exists, record why and classify producer residue, medium debt,
native-readiness gap, or missing substrate surface.
```

Prototype rows are bridge exemplars, not standalone ecology successes. Each
prototype must answer:

```text
what ecology demand does this serve?
which N05-N28 capability supplies it?
what bridge motif does it instantiate?
what composition, if any, makes it useful?
what debt remains before ecology can use it freely?
what exact next probe should agentic ecology run?
```

Prototype classes:

```text
trace / aftereffect prototype
reserve / pressure prototype
boundary / mobile-expression prototype
closed-loop perturbation prototype
proxy-collapse prototype
transfer prototype
generative/extractive medium-reshaping prototype
composition prototype
```

Prototype rows must state:

```text
source_artifacts
runtime_or_reconstruction_status
ecology_demand_role
supplied_capability
bridge_motif
bridge_exemplar_role
composition_role
next_probe_contract
agency_diagnostic_role
producer_residue
medium_debt
claim_ceiling
unsafe_claim_flags
```

A prototype may support:

```text
source-backed ecology-bridge structure
first-probe contract
producer-mediated prototype candidate
missing-native-mechanism evidence
```

It should not be interpreted as extra proof of the originating primitive unless
the relevant primitive experiment already supports that claim. Its main use is
to make a downstream ecology composition concrete and auditable.

A prototype must not support by itself:

```text
native ant agency
native colony agency
biological agency
organism/life
consciousness
sentience
semantic goals
fully native shared-medium coordination
```
