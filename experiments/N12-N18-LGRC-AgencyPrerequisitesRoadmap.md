# N12-N18 LGRC Agency Prerequisites Roadmap

This roadmap records the planned continuation after N11. It is a roadmap for
agency prerequisites, not for agency claims.

N11 closed the prior foundation arc at:

```text
A7 / GALI7
broader_general_artifact_only_agentic_like_integration_candidate
fully_native = false
```

The next tranche asks whether the artifact-only A7 foundation can move toward
endogenous, self-maintaining, consequence-sensitive LGRC dynamics while keeping
the claim boundary intact.

The central shift is:

```text
N05-N11:
    can the pieces compose and generalize artifact-only?

N12-N18:
    can the composed system naturalize its support, regulation, choice, and
    action loops without hidden producer scaffolding?
```

## Methodological Sources

N12-N18 should continue to use the Arc of Becoming methodology. These sources
are referenced by title only because they live outside this repository:

```text
Classification of Becoming
Interrogation of Becoming
Naturalization of Becoming
Cultivation of Becoming
```

The methodological emphasis changes slightly after N11:

```text
Classification:
    classify whether an apparent agency-like property is local observation,
    reusable becoming class, support-dependent expression, endogenous
    precondition, or native regime expression.

Interrogation:
    ask what boundary the system actually crossed: trigger, selection,
    regulation, self-maintenance, consequence sensitivity, self/environment
    separation, or closure.

Naturalization:
    test whether producer-supplied decisions become native runtime-visible
    substrate policies.

Cultivation:
    grow the missing support function instead of forcing a yes/no agency label.
```

## Claim Boundary

The following implications remain invalid:

```text
self-maintenance proxy => agency
consequence-sensitive route choice => intention
endogenous proxy formation => semantic goal ownership
self/environment boundary => selfhood or personhood
closed action-perception loop => organism or biological behavior
long-horizon closure => unrestricted agency
native policy support => semantic agency
```

The strongest conservative target for this tranche is:

```text
native_lgrc_agentic_like_dynamics_candidate
```

not:

```text
agent
agency
intention
semantic goal ownership
identity acceptance
personhood
biological behavior
unrestricted agency
```

## Agency-Prerequisite Ladder

This tranche should introduce a new local ladder, separate from the N11 GALI
ladder. A provisional ladder:

```text
AP0:
    passive integrated replay; no self-maintenance or consequence sensitivity

AP1:
    runtime-visible trigger produces bounded response

AP2:
    support-sensitive regulation preserves a declared support condition

AP3:
    self-maintenance candidate: regulation targets support/identity conditions
    rather than only an externally declared proxy

AP4:
    consequence-sensitive selection: route/action selection depends on expected
    downstream support, memory, or regulation effects

AP5:
    endogenous proxy candidate: proxy/target conditions are generated from
    runtime support or identity state, not supplied only by the experiment

AP6:
    self/environment boundary candidate: internal support-relevant state and
    external resource/perturbation state are separable in artifacts and controls

AP7:
    closed action-perception loop candidate: selected actions alter substrate or
    environment state, and those altered conditions feed back into later
    selection

AP8:
    long-horizon agentic-like closure candidate: identity/support, memory,
    regulation, consequence-sensitive selection, and action-perception feedback
    remain source-current, budget-clean, and claim-clean under longer windows
    and perturbations
```

`AP8` is not agency. It is the strongest agency-prerequisite evidence target
for this tranche.

## N12. Native Naturalization / Producer Dissolution

Question:

```text
Which N11 producer-mediated pieces can be replaced by native LGRC policy
without changing the result?
```

Primary input:

```text
N11 Iteration 11 native gap inventory
N11 Iteration 12 final closeout
```

Focus:

```text
route conductance memory
response magnitude regulation
cross-cutting budget / artifact replay contract
```

Expected output:

```text
native naturalization inventory
Phase 8-ready policy contracts
producer decision vs bookkeeping split
controls proving no hidden producer mutation
```

N12 should not try to prove agency. It should decide which mechanisms are ready
for native absorption and which remain experiment-local.

## N13. Self-Maintenance / Support-Seeking Regulation

Question:

```text
Can the system regulate to preserve its own support or identity conditions,
not only an externally declared proxy?
```

Primary inputs:

```text
N07 support / identity invariance evidence
N09 goal-proxy regulation
N10 support-sensitive integration
N11 GALI7 generalization envelope
```

Key distinction:

```text
external proxy regulation:
    target condition is declared by the experiment

self-maintenance candidate:
    regulated condition is derived from support/identity state
```

Expected controls:

```text
external-proxy relabel blocked
support-disrupted regulation blocked
identity-acceptance relabel blocked
budget ambiguity blocked
hidden support target blocked
```

N13 is probably the most important agency-facing experiment in this tranche.
Without self-maintenance, later agency-like claims are weak.

N13 closeout:

```text

phase8_opened = false
```

N13 supports an artifact-level self-maintenance candidate in the narrow AP3
sense: source-current support-seeking regulation distinguished from external
proxy regulation, stress-clean, budgeted, replayable, and claim-clean. It does
not prove agency, identity acceptance, intention, selfhood, personhood,
biological behavior, native support, or fully native integration.

## N15. Endogenous Proxy Formation

Question:

```text
Can proxy or target conditions arise from runtime support/identity state rather
than being declared externally?
```

This is the goal-ownership boundary.

N15 should distinguish:

```text
declared proxy:
    experiment supplies target band or target condition

endogenous proxy candidate:
    target condition is generated from runtime-visible support, identity,
    memory, or regulation state
```

Expected controls:

```text
externally injected target blocked
hidden target derivation blocked
semantic goal ownership relabel blocked
unbounded target drift blocked
budget-surface ambiguity blocked
```

Passing N15 would not prove goal ownership. It would prove that target
formation can be runtime-derived under bounded conditions.

## N16. Self / Environment Boundary

Question:

```text
Can the system distinguish support-relevant internal state from external
resource or perturbation state?
```

This is not a selfhood or personhood claim.

It asks whether artifacts can separate:

```text
internal support / identity state
external resource state
external perturbation state
action-mediated changes between them
```

Expected controls:

```text
internal/external label injection blocked
resource relabel as self blocked
self-support relabel as environment blocked
untracked boundary crossing blocked
identity acceptance relabel blocked
```

N16 should be careful: "self/environment" is an operational boundary in the
artifact chain, not a semantic self.

## N17. Closed Action-Perception Loop

Question:

```text
Can selected actions alter substrate/environment state, and can those altered
conditions feed back into later selection?
```

Target loop:

```text
runtime-visible evidence
-> route/action selection
-> scheduled action
-> substrate/environment consequence
-> artifact-visible changed condition
-> later selection consumes changed condition
```

Expected controls:

```text
action consequence missing blocked
stale consequence read blocked
producer direct mutation blocked
environment change without action blocked
semantic agency relabel blocked
```

N17 is where the behavior becomes loop-shaped. It still does not prove agency.

## N18. Long-Horizon Agentic-Like Closure Stress Test

Question:

```text
Can identity/support, memory, regulation, consequence-sensitive selection, and
action-perception feedback remain closed over long horizons and perturbations?
```

This should consume N12-N17 only if their records are source-backed and
claim-clean.

Expected stress dimensions:

```text
longer horizon windows
reversed route/context variants
support withdrawal and restoration
proxy perturbation
memory relaxation
environment/resource perturbation
duplicate/replay controls
artifact-only reconstruction
```

Expected result:

```text
native_lgrc_agentic_like_dynamics_candidate
```

only if native support has actually been implemented and validated. Otherwise
the result should remain:

```text
artifact_only_agentic_like_closure_candidate
```

## Native Absorption Interface

N12-N18 should coordinate with Phase 8, but not silently modify core LGRC.

Use this rule:

```text
experiment:
    discovers and classifies the needed mechanism

Phase 8:
    implements selected native LGRC mechanism in src/*

return experiment:
    reruns source-backed validators and controls using the new native support
```

Immediate Phase 8-facing candidates inherited from N11:

```text
native_route_conductance_memory_policy
native_response_magnitude_policy
```

Deferred or theory-sensitive candidates:

```text
native_identity_acceptance_validator
native_agentic_like_integration_policy
semantic goal ownership validator
native self/environment boundary validator
```

## Roadmap Acceptance

This roadmap is useful only if later experiments preserve these rules:

```text
each experiment has its own README, plan, checklist, outputs, reports, controls
each ladder has first-class inventory rows
claim flags remain separate from evidence classifications
artifact-only replay is required for closeout where applicable
negative controls fail with distinct blockers
producer-mediated success is not relabeled as native support
native support is not claimed without Phase 8 implementation and validators
agency/personhood/biology/unrestricted claims remain blocked
```

The intended progression is toward native agentic-like dynamics candidates, not
toward premature agency labels.
