# Hypothesis B - Proxy Divergence And Proxy Collapse

N26 can show that proxy improvement and basin deepening can separate in a
source-current, replay-backed, control-clean way.

Support requires:

```text
proxy_vs_basin_delta_trace present
proxy improvement measured under a declared metric
basin persistence capacity measured independently
support/coherence floors preserved or failure recorded
basin-deepening comparison trace present
peer/control basin trace present where multi-basin comparison is used
proxy divergence row shows proxy improves while basin persistence stalls or degrades
proxy collapse row shows proxy-optimized success fails under perturbation
basin-deepened contrast survives the same perturbation envelope
artifact replay, snapshot/load replay, duplicate replay, and order controls pass
negative controls fail closed
```

Failure conditions:

```text
proxy improvement is also basin deepening
basin degradation hidden by proxy scoring
perturbation changes between proxy-optimized and basin-deepened rows
collapse row lacks a basin-deepened survivor contrast
peer/control basin missing when required
post-hoc target formation
hidden proxy policy
proxy-only improvement relabeled as primitive success
```

Expected ceiling:

```text
PD4 controlled proxy divergence candidate
PD5 controlled proxy collapse candidate
PD6 only after replay, controls, AP5 gate, claim boundary, and N27 handoff pass
```

This hypothesis does not support semantic learning, semantic choice, agency,
native support, or goal ownership.
