# N26 Hypotheses

N26 tests proxy divergence / proxy collapse as a becoming primitive on scoped
multi-basin LGRC substrate.

Hypothesis set:

```text
Hypothesis A - Source-Current Proxy Derivation
Hypothesis B - Proxy Divergence And Proxy Collapse
Hypothesis C - Claim Boundary And Scoped Substrate Discipline
```

Hypotheses A and B may pass only if Hypothesis C also passes. If Hypothesis C
fails, any local proxy result is demoted or blocked regardless of margin.
