# N26 Configs

Configuration fixtures for N26 belong here.

Configs should declare proxy metrics, target digests, lower-stack input
windows, perturbation schedules, replay policies, and scoped MB6 substrate
selection before outcomes are inspected.

No config may use local absolute paths or relabel N25.2 scoped MB6 evidence as
unscoped multi-basin substrate, native support, agency, sentience, or ant
ecology implementation.
