# N24 Hypotheses

N24 tests abundance as surplus-supported optionality, not reward
maximization, semantic choice, or agency.

Hypotheses:

```text
Hypothesis A = source-current surplus above maintenance floors
Hypothesis B = optional continuation space under preserved basin integrity
Hypothesis C = claim boundary, producer residue, and AP-gap discipline
```

Hypothesis A or B may pass only if Hypothesis C also passes.
