# N25 Hypotheses

N25 tests spark / sub-basin / new-basin formation, not semantic learning,
choice, agency, or native support.

Hypotheses:

```text
Hypothesis A = native source-current basin-formation evidence
Hypothesis B = producer-assisted flux-conditioned formation scaffold
Hypothesis C = claim boundary, lane separation, and control discipline
```

Hypothesis A or B may pass only if Hypothesis C also passes.
